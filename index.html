concept

The concept of this project was for us to be able to quickly and easily program any size led matrix to display an image.

motivation

My teammate and I are both on FRC Team 5417, and our robots cart, as well as the robot itself, is very plain, so we decided to spruce it up with an led matrix display, however we wanted to be able to easily switch it to display things such as alliance color and our logo. Because of this we decided to use an led matrix, and write some code to make it less labor intensive to convert bitmaps into instructions for the matrix.

code, and how it works

This python script takes a bitmap image (in this case pxartT.bmp) and iterates through the bitmap using a for loop to iterate through the y-coordinates, printing arduino led commands as it goes. As it loops it increases the led index (z), and when each column of y coordinates ends, the for loop it is nested inside shifts the x-coordinate and reverses the direction of the change in y-coordinates to imitate the snake-like pattern of the led matrix.

After the python script is run, it echoes the led arduino commands to the shell, which is then copied into the template shown above, and pushed to the arduino.

conclusion

While the code and mechanics of the project were relatively simple, an unexpected challenge was debugging an issue we had, which was powering the entire 3 matrices, was the main issue. We eventually found out was due to a lack of memory in the arduinos, and was caused by our lack of knowledge in arduinos.
